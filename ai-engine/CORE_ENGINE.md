# Core Engine

The GPU Energy Optimizer core engine is proprietary and not publicly distributed.

This includes:
- engine1_efficiency.py through engine8_delta.py
- optimizer.py
- gpu_monitor_agent.py
- database.py
- metrics_simulator.py

## What is available
- All benchmark results in /data
- CEI methodology in /docs
- Reference implementation in /reference
- Frontend dashboard in /app

## Verify Results
CEI = Total FLOPs / Total Joules

All raw data is available in /data/tests to verify independently.
