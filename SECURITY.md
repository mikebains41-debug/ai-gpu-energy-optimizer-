# Security & Compliance Status

GPU Optimizer Inc. maintains active security hygiene practices, including automated dependency vulnerability scanning (GitHub Dependabot) and static code analysis (CodeQL) across our codebase, with continuous monitoring for known vulnerabilities and exposed secrets. 

We are currently pursuing SOC 2 Type II and ISO 27001 certification; both are in progress, not yet complete. 

For vulnerability reports or compliance inquiries, contact the founder team directly.
